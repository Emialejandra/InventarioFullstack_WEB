#  Proyecto Fullstack - Supabase y Firebase Auth

Aplicación web fullstack desarrollada con autenticación de usuarios utilizando **Supabase Auth** y **Firebase Authentication**, integrando frontend y backend mediante API REST.

El proyecto incluye:
- Registro e inicio de sesión de usuarios
- Confirmación de correo electrónico
- Backend funcionando correctamente
- Dashboard de productos
- Integración con PostgreSQL
- Manejo de autenticación moderna
